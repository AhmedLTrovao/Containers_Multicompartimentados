Neste projeto de pesquisa será estudada uma extensão do problema de carregamento de
contêineres em que o contêiner é composto por vários compartimentos com acesso lateral. Neste
problema, clientes devem ser visitados por um contêiner (caminhão) em uma rota predefinida, e as
caixas demandadas por estes clientes devem ser arranjadas sem sobreposição e completamente
dentro dos compartimentos deste contêiner. O objetivo consiste em maximizar o volume (ou valor)
das caixas empacotadas dentro do contêiner satisfazendo uma série de restrições práticas. Algumas
aplicações reais deste problema podem incluir aquelas relacionadas à logística de distribuição de
mercadorias, como, por exemplo, a distribuição de bebidas, mas também situações em que caixas
necessitem estar separadas ou agrupadas dentro do contêiner, no que torna-se conveniente o uso de
um contêiner com múltiplos compartimentos. Trata-se de um problema de otimização combinatória
de difícil solução exata, e, até onde se tem conhecimento, há apenas um trabalho na literatura que
tratou este problema e que propôs modelos de programação linear inteira mista bastante específicos
para descrevê-lo. Neste projeto de pesquisa pretende-se desenvolver novos modelos de
programação linear inteira mista, que permitam arranjos mais gerais que os existentes na literatura,
para descrever este problema, implementá-los utilizando linguagens de modelagem e aplicativos de
otimização, e realizar experimentos computacionais com exemplos gerados aleatoriamente e
baseados na literatura disponível.
